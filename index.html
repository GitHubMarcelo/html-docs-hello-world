Azure App Service Web Apps

Service Web Apps é uma plataforma de computação totalmente gerenciada que é otimizada para hospedagem de sites e aplicativos web. Essa oferta de plataforma-como-serviço (PaaS) do Microsoft Azure permite que você se concentre em sua lógica de negócios, enquanto a Azure cuida da infraestrutura para executar e dimensionar seus aplicativos.

Azure Content Delivery Network (CDN)

Content Delivery Network (CDN) armazena conteúdo estático da Web em locais estrategicamente posicionados para fornecer alta taxa de transferência para entregar conteúdo aos usuários. O CDN oferece aos desenvolvedores uma solução global para fornecer conteúdo de alta largura de banda armazenando em cache o conteúdo em nós físicos em todo o mundo.
